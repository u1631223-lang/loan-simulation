# 住宅ローン電卓 ユーザーガイド

商談中の暗算をゼロにする業務向け電卓の使い方をまとめています。
本番 URL: https://loan-simulation-eight.vercel.app

## 1. 起動して最初にやること

1. 上記 URL を開く
2. iPad / iPhone：Safari の「共有」→「ホーム画面に追加」
3. Mac / Windows：Chrome のアドレスバー右側にあるインストールアイコンを押す
4. 以降はホーム画面アイコンから起動 → **前回の入力が自動復元**されます

## 2. ローン計算（forward）

借入額・金利・期間・返済方式・ボーナス払いを入力し「計算する」を押すと月々返済・総返済・利息が表示されます。

- 上下ボタンで微調整：借入額 100万円／金利 0.01%／年 1年／月 1ヶ月
- ボーナス月は 1 月・8 月固定
- 「金利上昇シナリオ比較」で +0.5%／+1.0%／+1.5% の結果を一括比較

## 3. 逆算モード（reverse）

月々返済額から借入可能額を算出します。お客様が「月◯万円までなら払える」と仰った時に最適です。

## 4. 年収 MAX 借入額（income）

年収を入れると、返済負担率 30%／35% を年収 400 万円基準で自動判定し、借入可能額を提示します。連帯債務／連帯保証も 100% / 50% 合算で対応。

「詳しい返済計画を立てる」ボタンで、計算結果を forward モードに引き継いで詳細シミュレーションへ移動できます。

## 5. 返済負担率モード（repayment-ratio）

年収と希望返済額から負担率を直接算出します。

## 6. 繰上返済（/loan-tools）

期間短縮型／返済額軽減型を選び、繰上時期と金額を入れると効果（節約利息・短縮期間）が出ます。

## 7. ローン比較（/loan-tools）

最大 5 件のローン条件を一括比較。総返済額・利息・月々返済の差分を表で確認。

## 8. NISA 複利

月々積立 + 初期投資 + 想定利回り + 期間で資産推移をシミュレーション。デフォルト利回りは S&P500 の長期平均（約 10.5%）の保守的見積もりとして 7%、期間 40 年です。

## 9. 簡易電卓

商談中の坪計算・税込計算用。M+/M-/MR/MC のメモリ機能、計算履歴も搭載。

## 10. 履歴・エクスポート

- 履歴は最大 20 件まで自動保存（端末ローカル）
- 返済予定表は CSV／Excel エクスポート可
- PDF はローン結果 + 返済予定表抜粋（最初の 12 ヶ月＋最後の 12 ヶ月）を出力

## 11. 入力をリセットしたい

「初期値に戻す」ボタンを押すか、ブラウザの「サイトデータをクリア」で全削除できます。

## 12. プライバシー

すべての入力値・履歴はご利用端末のブラウザ内に閉じています。外部 API への通信はありません。詳細は [プライバシーポリシー](../public/privacy-policy.md)。
